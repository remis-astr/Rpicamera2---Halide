# ============================================================
# NOTES D'UTILISATION :
# ============================================================
#
# 1. Pour activer la calibration automatique par histogrammes:
#    - isp_enable : 1
#    - isp_auto_calibrate_method : 1
#    - isp_auto_calibrate_after : 10 (ou plus)
#
# 2. Pour des sessions courtes (< 100 frames):
#    - isp_recalibrate_interval : 0 (pas de recalibration)
#
# 3. Pour des sessions longues (> 200 frames):
#    - isp_recalibrate_interval : 50 (recalibrer tous les 50 frames)
#
# 4. Pour désactiver complètement la calibration auto:
#    - isp_auto_calibrate_method : 0
#
# 5. Voir CALIBRATION_AUTO_ISP.md pour plus de détails
#
# ============================================================
